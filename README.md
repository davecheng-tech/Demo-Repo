# Processing: Calling Functions

Make sure to complete your work in `src/Sketch.java`.

## Learning Objectives
In this task, you will learn about:
- Calling Processing functions
- Draw order
- Coordinate systems
- Colours

## Step 1: Lesson Notes
Review the following tutorial to learn the basics:  
[Happy Coding – Calling Functions](https://happycoding.io/tutorials/processing/calling-functions)

## Step 2: Task
Demonstrate your learning by creating a simple drawing in Processing. Write your code in the `src/Sketch.java` file.

Your drawing must include:
- A **background colour**
- At least **3 different shapes** (e.g., `rect`, `ellipse`, `line`, `triangle`, `polygon`)
- A clear **foreground** and **background**

### Example Ideas
- Draw a flower or a garden
- Draw a rainbow
- Draw a dog or cat
- Draw a house

Be creative and challenge yourself!

## Step 3: Submission
1. **Commit** and **Sync/Push** your code to your assignment repository.
2. Take a **screenshot** of your Processing program's graphic output.
3. Upload the screenshot to the **Google Classroom assignment post**.
4. Follow the [Java Style Guide](https://github.com/SACHSTech/Java-Style-Guide) for good coding practice.
